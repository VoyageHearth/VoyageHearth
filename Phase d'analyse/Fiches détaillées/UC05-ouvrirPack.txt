UC05 : Ouvrir pack

Auteur : VoyageHearth Team

Description : Le joueur ouvre un pack et se voit attribué les cartes
              
Acteur : Joueur connecté

Précondition : Aucune

Scénario nominal :

    1. Le joueur choisit l'option "ouvrir pack"
    2. Le système affiche les packs en attente d'ouverture
    3. Le joueur sélectionne un pack en attente
    4. Le système affiche les 5 cartes contenues dans le pack ouvert
    
Postconditions : Le système ajoute les cartes contenues dans le pack à la collection du joueur

Alternative :

    A1 : En SN3, le joueur sélectionne plusieurs packs disponibles
    
Exceptions :

    E1 : En SN2, aucun pack en attente d'ouverture
        
    E2 : En SN3, annulation de l'opération par le joueur
